Push/Pull

Push or Pull columns by adding the class "pull-X-span" to the column. Change the settings by expanding the class name with "-s0pl". S stands for the Grid-Size, 0 ist the value of Push/Pull and p stands for push, while pl stands for pull. You can also use "center" and "right" for positioning.

.col-12-span .push-6-span-s0p LEFT

.col-12-span .pull-2-span-s0pl RIGHT

.col-3-span .center

.col-3-span

.col-3-span .right

Custom Column Change

You have fully Control over the columns and their behaviour. Just expand the Column-Class like this example -l6c-m3c-s4c. The Letter is for the Grid-Size and the Number for the new column width. The c must be at the and of each one.

.col-2-span-l4c-m3c-s4c

More Options

Clear: If the columns not floating proberly, just ad clear-xl12-l12-m4-s3-xs2 to the Grid-Element. The Number stands for the column Elements per row.

Colpad: Ad "colpad" to the Grid-Element to give it an inner padding.

Border-Gutter: By adding the Class "border-gutter" to the Grid-Element, you can add a background-color directly on the column-Element and don't need an extra element.

Hide/Show: Ad for example "hidden-l" to an Element to hide it on Grid L. Show Elements by adding "visible-l".

Safari Subpixel Rendering: I included a Fix for the Subpixel-Rendering-Issue with Safari. It works fo unnested Columns.
